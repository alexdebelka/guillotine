# Consensus / literature search prompts

Consensus works best with **one focused research question per search** (not keywords). Set **year ≥ 2020**.

## Primary
```
What methods enable contrast-agnostic cross-modal retrieval of same-subject 3D brain MRI between contrast-enhanced T1 and T2?
```
